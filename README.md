# Order Management System

The purpose of this project is to create an order management system with a configurable plugin structure.
It should allow different configurations for different customers. Based on the different configurations,
the system should allow/disallow certain features.

## Plugins Implementation
For the implementation of the plugins, we are using FlagSmith with is an open source feature flagging service.
It allows us to configure different features for different customers.

## Run the app in development mode
`yarn run dev`

## Build the app for production
`yarn run build`

## Run the tests
`yarn test`

## Run the tests with coverage
`yarn run test:coverage`
